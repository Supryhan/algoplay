# Go Lab

This directory contains small Go examples used for learning, experimentation, and interview preparation.

The current structure is intentionally minimal.

## Structure

    go/
    ├── go.mod
    └── examples/
        └── basics/
            └── main.go

## Current Examples

- `examples/basics/main.go` — minimal working Go program.

## Run

From the repository root:

    cd go
    go run ./examples/basics
